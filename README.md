# Cấu trúc dự án

```
├── docs/                        
│   ├── SRS/                      # Đặc tả yêu cầu phần mềm (Software Requirement Specification)
│   ├── UseCase/                  # Use case + mô tả luồng nghiệp vụ
│   ├── Design/                   # Thiết kế hệ thống: UML, ERD, kiến trúc...
│   ├── TestCases/                # Kịch bản kiểm thử (test case)
│
├── src/                        
│   ├── backend/                  
│   │
│   ├── frontend/                 
│
├── database/                     # Cơ sở dữ liệu
│   ├── schema.sql                # Cấu trúc bảng: hộ khẩu, nhân khẩu, thu phí...
│   └── sample_data.sql           # Dữ liệu mẫu để test
```
